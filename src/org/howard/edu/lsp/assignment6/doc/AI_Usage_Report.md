# AI Usage Report

AI assistance was used in a limited and supportive role during the development of Assignment 6.

Specifically, AI was used to:
- Review the assignment requirements and ensure that all required methods were tested
- Identify required edge cases, including:
    - add() with duplicate values
    - remove() when the value is not present
    - union() with an empty set
    - intersect() with no common elements
    - diff() with identical sets
    - complement() with disjoint sets
    - largest() and smallest() on an empty set using assertThrows
- Assist in understanding how to properly structure JUnit 5 test cases
- Troubleshoot issues related to IntelliJ setup and JUnit configuration so that tests could run successfully
- Verify correctness of test coverage against the assignment rubric

AI was not used to blindly generate the final solution. All code was:
- Written, reviewed, and tested manually
- Adjusted to match the assignment requirements
- Executed and validated within IntelliJ

The final implementation and test cases represent my own understanding of the material and were verified through successful execution of all test cases.
